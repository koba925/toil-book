# toil-book

## このリポジトリについて

このリポジトリは、Zennにて執筆予定の『100行のPythonから育てて学ぶ プログラミング言語の作り方』（仮題）のためのソースコードです。
章ごとにプログラミング言語（コード）が成長していく過程を、Gitのタグを使って簡単に追うことができます。
章ごとにタグが打ってありますので、GitHubの「Tags」で見たい章のタグ（例：0101_constants）を選択してください。

現在は第一部となる予定の、最小のインタプリタのソースコードまでが入っています。
最小のインタプリタとはいえ、500行程度のPythonでツリーウォーク型インタプリタと中間コード＋仮想マシンによるインタプリタの両方が体験できるのがちょっと自慢です。

## サンプルコード

たとえば以下のようなコードが実行できます。

```
def factorial_iter(n) do
    result := 1;
    while n > 0 do
        result = result * n;
        n = n - 1
    end;
    result
end
```

その他の実行例については [0312_examples/toil.py](https://github.com/koba925/toil-book/blob/0312_examples/toil.py#L527) の下部、`# Example`のコメント以下をご覧ください。
Noneとboolとintしかなくて配列もタプルも辞書もクラスもないのになぜか二分探索木まで実装できています。ふしぎ！

## ⚠️ 注意事項
このリポジトリは、執筆の都合上、ソースに変更があった場合**予告なく履歴の書き換え（Force Push）を行います**。

そのため、お手元に `git clone` すると、後日 `git pull` を行った際にコンフリクト（競合）エラーが発生する可能性が高いです。
コードをお手元で確認・実行したい場合は、「Download ZIP」からダウンロードしていただくか、cloneした場合はpullせず、更新したいときには再度cloneからやりなおすことをお勧めします。
